# Code of Conduct

Silrahi is an open contribution project. We want contributors to feel welcome, respected, and safe.

## Our Standards

Expected behavior:

- Be respectful and patient.
- Give constructive feedback.
- Assume good intent, but accept correction when impact differs from intent.
- Keep discussions focused on the project and its users.

Unacceptable behavior:

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Sharing private information without permission.
- Repeatedly derailing issues or pull requests.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or contributions that violate this Code of Conduct. Repeated or severe violations may lead to temporary or permanent bans from project spaces.

## Reporting

Report conduct concerns privately to the maintainers. If the project enables GitHub private vulnerability or contact channels, prefer those for sensitive reports.
